# Configuration

Environment variables and runtime configuration.

## Key Rules

- Set `LINEAR_API_KEY` to skip all interactive auth setup.
- Set `NO_COLOR` (any value) to disable all ANSI output.
- Cache: `$XDG_CACHE_HOME/linear/`, credentials: `$XDG_CONFIG_HOME/linear/credentials`.
